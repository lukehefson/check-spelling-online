# Installation

1. Clone/download it
2. Open `SpellCheckService.xcodeproj` in Xcode
3. Build and run (cmd+R) the app once in Xcode to register the service with macOS
4. Quit the running app/quit Xcode
5. Restart your Mac
6. See the screenshot above for how to use it
